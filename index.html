GitHub Sync

On startup the vault automatically loads any new presets from prompt-vault-export.json in your GitHub repo. To publish your local presets: export JSON below, then upload it to your repo root.

Export / Import Presets

GitHub GIF Repo

Upload GIF files to github.com/Gerardosauro/prompt-vault/tree/main/gifs then enter just the filename (e.g. my-video.gif) in the preset form. Upload your presets JSON as prompt-vault-export.json to the repo root to enable auto-sync.
